Binding Your Own Keys

To run the Default Bindings Editor left click on the Default Bindings tree item in Game Manager.

To add or edit a binding simply double click it. This displays the Choose Bind Command dialog box.

Double clicking a Bind Command from this dialog sets the binding in Default Bindings Editor. For example, to set the key W as the Move Camera Forward command, you would double click the KEY_W item, then double click Camera.Move.Forward binding. The main view would now appear as follows:

That's pretty much it for setting bindings. An exception is the Camera.Current.Set command which requires you to identify a Camera to set. For example, if you had 3 cameras in your game named, say, Cam0, Cam1 and Cam2, you could make the keys F1, F2 and F3 set them as the current camera each time you pressed these keys on your keyboard. You would do as follows to achieve this:

Double click the item KEY_F1, this opens the Choose Bind Command dialog box, double click the item Camera.Current.Set, this displays the following Choose Camera dialog:

Double click the Cam0 item (obviously you would have to add the 3 cameras prior to setting this binding). You'd repeat this process for bindings to the keys F2 and F3 to cameras Cam1 and Cam2 respectively. After this is done, your main view would appear as follows:
