Applying texture sets to your map

Applying Texture Sets to your map is done using the Brush technique and the Selection Block technique. For information on how these work see the Selection block vs brush editing tutorial. The same principals apply to Texture Set assignment as they do to Cell insertion and removal.

The user must create Texture Sets before they can be applied to the map. How to do this is covered in the Basic Map Creation tutorial.

With some Texture Sets created, choose the Texture Set item from the Edit Mode droplist towards the top of the Cell Map Editor window.

The Texture Set dialog bar will be shown docked to the right side of the Cell Map Editor. It appears as follows when the Current Brush is set to Select:

The Default Texture Set droplist controls which Texture Set is used when you insert new cells into the map. Changing the Current Brush droplist item will cause the Texture Set dialog bar to change which controls appear inside it. This will be covered below.

Texture Set assignment via Selection Blocks

With the Current Brush droplist set to Select, you may create a Selection Block in the map editor window that encompasses one or more cells of your map. Check the Selection block vs brush editing tutorial if you're not sure how to do this. Once created, click the Set Texture Set button and choose one of the Texture Sets. Texture Set assignment via Brushes

To apply Texture Sets via the Brush technique change the Current Brush on the Texture Set dialog bar to Apply, shown here:

Choose any Texture Set from the list box, and left click and drag on cells in the map to apply the Texture Set. You'll notice door cells and internal wall cells adopt the texture set you apply if you apply it to the cell next to them. Door cells are considered to be Normal Doors, Secret Doors and Translucent doors. Internal Wall cells are considered to be Translucent Small Walls and Translucent Large Walls.
