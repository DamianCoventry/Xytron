Dynamic enabling/disabling of collision detection

If you've read the Binding your own keys tutorial then achieving dynamic enabling/disabling of collision detection might already be obvious to you. Simply create a binding to the to Camera.ToggleCollision command using the Default Bindings Editor.

Here's the steps:

  1. Left click on the Default Bindings Editor in the Game Manager
  2. Double click an Input Event that you want to make toggle collision detection on/off (this example uses KEY_C)
  3. Double click the Camera.ToggleCollision command
And that's it. When you run your map in SageEngine you may press the C key to toggle camera collision detection on and off.
