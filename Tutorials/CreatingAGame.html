Creating a game

Start SageStudio via any of the conventional means:

Double click the program in ExplorerDouble click the desktop IconClick the Start Menu item

When SageStudio starts it presents you with an empty studio window:

Create a new game by clicking New from the File menu, or by pressing Ctrl+N. SageStudio will present you with a dialog box asking you to enter a name. For this example I called my game PenguinShooter. After dismissing the dialog box by clicking OK, SageStudio creates a new empty game for you, as shown here:
