Moving Around the Map

Moving the viewpoint

Moving around the Cell Map Editor is achieved via the right mouse button. Simply right click the mouse button within the editor window to grab the window, then drag the mouse cursor to move the window view point around.

Zooming in/out

When the input focus is in the window you may use the mouse scroll wheel to zoom in or out. To set the input focus to the editor window, simply click it with either mouse button.

Centering the view point

To center the view point on a cell double right click on any point in the editor window.

Rotating the view point

This is achieved by pressing and holding either shift key, then right clicking the mouse button within the editor window to grab the window, finally, drag the mouse cursor left or right to rotate the view point.
