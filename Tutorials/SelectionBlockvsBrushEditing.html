Selection block vs brush editing

The two ways to add and remove cells from the world are using a Selection Block and using a Brush. Both have their pros and cons and therefore both were added to allow the user to decide how best to create their map.

Brushes

The brush technique involves inserting a single cell at a time via the brush concept. Much like painting a canvas in real life, this concept revolves around the idea of allowing the user to pick up an object and drag it over the world. The object you pick up, is called a brush. The brushes available in SageStudio are as follows:

  • Select
  • Solid Space
  • Normal
  • Secret
  • Pit
  • Liquid
  • Door
  • Secret Door
  • Translucent Door
  • Translucent Small Wall
  • Translucent Large Wall

The Select brush is the odd one out, and is covered next in the Selection Blocks section. Every other brush adds or removes a type of cell to/from the world as you click and drag your mouse cursor over the editing window.

To edit your map via the brush technique, left click on a map in the Game Manager and choose Cell from the Edit Mode droplist towards the top of the Map Window. The Current Brush droplist on the right docked dialog bar controls which brush you're currently holding. Each brush type is covered next, except for the Select brush, which is in the following section.

Solid Space Brush
This brush removes cells from the world
Normal
The ubiquitous normal cell that forms the basis of every room and hallway is created by this Brush. This cell has four walls, a ceiling and a floor and the player can pass through it without obstruction. Wall polygons are removed when this cell is placed next to another cell.
Secret
This brush inserts Secret cells, that are identical to Normal cells except they're flagged as secret and not drawn on the player's map until discovered by the player.
Pit
This brush inserts cells that have a floor 10 cells deep. If a player walks into this cell they will drop to their death.
Liquid
This cell is one cell deep and filled with liquid; typically water but it can be any liquid if you change the liquid texture and color. The player's viscosity is modified when travelling through this cell, and buoyancy is simulated by SageEngine. The player cannot hold their breath forever if fully submerged in the liquid, and will drown if they're not careful.
Door
A basic door cell is inserted by this brush. The player cannot travel through this cell until they use their action key to open the door. The door raises upwards and the player may move through afterwards. This cell adopts the textures of its neighbouring cells for its walls, ceiling and floor. However the door texture can be set explicitly.
Secret Door
This brush inserts a Secret Door, which is a door that fills the entire cell and is not drawn on the player's map until discovered. This door itself adopts the textures of its neighbouring cells so that you can't tell it's there. The player must use their action key on the door to discover its presence.
Translucent Door
This brush inserts a Translucent Door, which is exactly the same as a normal door, except that you can see through it.
Translucent Small Wall
This brush inserts an internal wall, which is a wall that passes through the middle of the cell. A player cannot pass through this cell. The cell has a small window in it that is translucent. The window's glass can be destroyed with the action key. This cell adopts the textures of its neighbouring cells for its walls, ceiling and floor. However the window texture can be set explicitly.
Translucent Large Wall
This brush inserts an internal wall that is similar to the small translucent wall. The differences are that the window fills the entire cell, and once destroyed with the action key, the player can pass through this cell.

Here's a simple example of creating a map with 3 rooms and 3 doors. For variety, pit and liquid cells were added. This was done by selecting the appropriate cell brush and left clicking in the editor window, and dragging the cursor across the window. If a mistake was made, the solid space brush was used to remove cells from the world, using the similar click and drag technique.

Selection Blocks

Selection Blocks provide a convenient way to insert many cells at once. Choose the select brush from the Current Brush droplist on the right docked dialog bar. To create a selection block left click on the editor window and drag the cursor across the window, as shown here:

Another way to create the same Selection Block is to left click once on your map, then move the mouse cursor to another part of you map, and press and hold either shift key and left click the mouse button again. This achieves exactly the same thing as the click and drag technique.

Another feature of the Selection Block mechanism is that you can add and remove cells from Selection Blocks using by pressing and holding either control key and left clicking on the map. This next screen shot shows a Selection Block that's been modified by multiple control left clicks:

The final feature with Selection Blocks is that you can control left click drag to add to a selection block. That is, left click and drag across the map to create a Selection Block. Next, press and hold either control key, then left click and drag across the map to add another dragged set of cells to your Selection Block. The results of this technique is shown here:

Once you've created a Selection Block, you may apply a brush to every cell in the selection. This is achieved by clicking the button Set Cell Type on the right docked dialog bar, and then double clicking a cell type from the Choose Cell Type dialog box. Notice that you can choose the Solid Space brush from this dialog box too, which means you can remove entire blocks of cells from the world in one go.

Using the Set Texture Set button can be used also to apply a texture set to the Selection Block too, which is also another great way to speed up Texture Set assignment.
